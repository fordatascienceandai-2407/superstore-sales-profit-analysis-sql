# Superstore Sales & Profit Analysis using SQL

## Project Overview

This project analyzes Superstore sales and profit data using PostgreSQL.

The objective of this project is to solve real-world business problems using:

- SQL Queries
- Common Table Expressions (CTEs)
- Window Functions
- Ranking Functions
- Views
- Aggregations
- Business Analysis

## Dataset

The dataset contains information about:

- Orders
- Customers
- Products
- Categories
- Regions
- Sales
- Profit

## Business Questions Solved

1. Top Customers by Sales
2. Top Products by Category
3. Regional Sales Contribution
4. Customer Segmentation
5. Monthly Sales Trend Analysis
6. Running Totals
7. Moving Averages
8. Profit Ranking by Region
9. Sales Bucketing using NTILE
10. Advanced Window Function Analysis

## SQL Concepts Used

- GROUP BY
- HAVING
- CASE WHEN
- CTEs
- Views
- ROW_NUMBER()
- RANK()
- DENSE_RANK()
- NTILE()
- LAG()
- SUM() OVER()
- AVG() OVER()
